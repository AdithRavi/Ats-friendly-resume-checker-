# ATS Resume Checker

An AI-powered web app that analyzes resumes, calculates ATS scores, and provides suggestions to improve job selection chances.

## 🚀 Features
- Resume Upload (PDF/DOCX)
- ATS Score (0–100)
- Keyword Matching
- Missing Skills Detection
- Improvement Suggestions

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js / Flask
- NLP: spaCy / NLTK

## ⚙️ How It Works
1. Upload resume  
2. Enter job description  
3. Analyze keywords & skills  
4. Get ATS score + suggestions
